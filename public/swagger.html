GET /api/v1/use_callback_code

Tags: Login

The APP needs a temporary token to use to communicate with the API. The API uses the Google user's email address to create this token. However, the API does not trust the APP. So after retrieving an email address from Google, the API returns an unencrypted email address the APP can use to identify the user, and a time-limited JWT containing the encrypted email address the APP can use as a bearer token to communicate with the API. This bearer token is how the API identifies the user until another bearer token is returned to the APP.
code

This is the code you get back from Google as part of the OAuth process. We will use this to get the user's credentials from Google.

 query string
200 OK

A JSON object containing two key-value pairs

Email

Schema definitions

Courses: object

id: number

A unique identifier for the Canvas course. Return this value to the API for subsequent course-related requests.

other_values: object

Other values the Canvas API provides including: name, account_id, ...

Email: object

encrypted_email: string (Base64)

A JWT that needs to be returned for the next few calls to the API.

email: string (email)

An email address the APP can use to identify the user

Tokens: object

obfuscated_canvas_token: string

Token from canvas but obfuscated such that it can safely be displayed

canvas_url: string

The URL the saved Canvas token can access

access_key: string

A pseudo-unique identifier of tokens you must return when you want query the Canvas API
